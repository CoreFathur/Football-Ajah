# Football Analyst AI

A private, personal football analysis platform. This initial commit contains a production-ready Next.js (App Router) + TypeScript + Tailwind skeleton with component-driven UI, placeholders, and architecture scaffolding.

Goals and constraints are documented in /docs.

What is included
- App router with pages: home, today, tomorrow, live, match/[id]
- Reusable components (MatchCard, Header, UI primitives)
- Services layer with stubs for data providers
- Types and utility functions
- Loading / Empty / Error states
- Dark, minimal UI theme

Next steps
- Wire a football data provider into services/matchService.ts
- Add AI provider integration for analysis and chat (providers/ai/*)
